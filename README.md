# Garden Dashboard v2

Νέα έκδοση με mobile-first σχεδιασμό εμπνευσμένο από σύγχρονες εφαρμογές φροντίδας φυτών.

## Κύριες ενότητες
- Πρόγραμμα: ημερολόγιο, καθημερινές και εβδομαδιαίες εργασίες, πότισμα.
- Διάγνωση: φωτογραφική καταγραφή συμπτωμάτων και ιστορικό.
- Παρτέρια: κάρτες των δύο παρτεριών και σημειώσεις.
- Εργαλεία: νερό, εδαφοκάλυψη, λίπανση, ψεκασμοί, κλάδεμα.
- Ιστορικό: φωτογραφίες και ημερήσιες καταχωρήσεις.

## iPhone
Η ενότητα «Εργαλεία» δημιουργεί αρχείο ICS για 12 εβδομάδες, καθημερινά στις 07:00, στη ζώνη Asia/Nicosia.

## Εγκατάσταση ως PWA
Ανέβασε τον φάκελο σε HTTPS hosting (GitHub Pages, Netlify ή Vercel), άνοιξε τη σελίδα στο Safari και επίλεξε:
Share → Add to Home Screen.

## Αποθήκευση
Τα δεδομένα και οι φωτογραφίες αποθηκεύονται τοπικά στον browser μέσω localStorage.
